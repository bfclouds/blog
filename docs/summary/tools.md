### itemr2命令提示

[zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions)

```sh
cd ~/.oh-my-zsh/plugins/
git clone https://github.com/zsh-users/zsh-autosuggestions
```

配置，编辑文件 ~/.zshrc

```sh
plugins=(
  git zsh-autosuggestions
)
```

```sh
source ~/.zshrc
```
